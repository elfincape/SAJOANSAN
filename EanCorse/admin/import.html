📥 코스표 일괄 가져오기

사용 방법

  1. 엑셀 시트를 UTF-8 CSV로 저장합니다 (엑셀 → 다른 이름으로 저장 → "CSV UTF-8").
  2. 아래 1단계에서 CSV 파일을 선택하면 자동으로 컬럼이 인식되고 미리보기가 표시됩니다.
  3. 2단계에서 컬럼 매핑을 확인/수정합니다 (헤더가 다를 경우).
  4. 3단계에서 정규화 결과를 검토합니다 (회사/기사/차량/납품처/코스/정거장).
  5. 4단계에서 "DB에 저장" 버튼을 눌러 일괄 upsert를 실행합니다.

⚠️ 같은 데이터를 여러 번 올려도 unique 키(회사명, 차량번호, 납품처 코드 등) 기준으로 upsert되므로 중복 생성되지 않습니다. 단, 정거장은 (route_id, stop_order) 기준 upsert되므로 기존 코스의 정거장이 덮어써질 수 있습니다.

1단계 · CSV 파일 선택

엑셀에서 헤더를 어떻게 정리해야 하나요?

1행에 다음 헤더(또는 유사 이름)가 있어야 합니다. 정확히 일치하지 않아도 2단계에서 수동 매핑 가능:

호차, 운수사, 코스명, 톤수, 차량번호, 운전자명, 연락처, 입차시간, 하차시작, 하차종료, 납품마감, 코드, 납품처명, 1톤이하, 3.5톤이하, 5톤이상, 무인야적납, 대면, 검수, 무인, 보안키, 열쇠, 없음, 창고, 탑차, 야적, 주소, 비고

2단계 · 컬럼 매핑 확인

CSV의 컬럼명을 시스템 필드에 연결합니다. 자동 인식되지 않은 항목은 직접 선택하세요.
