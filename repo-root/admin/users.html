🔐 사용자 추가 절차

  1. Supabase 대시보드의 Authentication → Users에서 "Add user"로 이메일·비밀번호를 직접 발급합니다.
  2. 생성된 사용자의 user_id (UUID)를 복사합니다.
  3. 아래 "신규 프로필 생성" 폼에 붙여넣고 표시이름·역할을 지정해 저장하면 시스템 사용자로 등록됩니다.

※ 이 화면은 user_profiles 테이블만 관리합니다. 이메일·비밀번호 자체의 변경/재설정은 Supabase 대시보드에서 수행하세요.

참고: 트리거가 설치되어 있으면(handle_new_auth_user) 1단계 직후 user_profiles에 빈 행이 자동 생성됩니다. 이 경우 신규 폼 대신 목록에서 바로 편집해 활성화하면 됩니다.
