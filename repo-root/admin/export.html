📤 코스표 내려받기

사용 방법

  1. 출력 옵션과 정렬 순서를 설정합니다 (기본: 기사명 ▲ → 하차종료시간 ▲).
  2. "미리보기 새로고침" 으로 결과를 확인합니다.
  3. "내려받기" 버튼으로 파일을 저장합니다.

⚠️ 출력 형식은 admin/import.html 의 입력 형식과 동일합니다. 수정 후 다시 import 하면 그대로 반영됩니다.
